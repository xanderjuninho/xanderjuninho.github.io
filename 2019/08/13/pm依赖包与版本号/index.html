npm查看本地包版本号和远程包的版本号

npm查看本地包版本号
npm语义化版本号

npm使用 a.b.c:
a为大版本号，有重大api改变，一般不向下兼容
b为小版本号，新增功能，向下兼容
c为补丁号，通常修复一些bug

npm安装包版本号中特殊符号意义:

更新所有 npm 依赖包

使用 npm-check 或 yarn

使用 npm-check 更新项目依赖
使用 yarn 更新项目依赖

npm-check 是检测 package.json 文件，项目存在 node_modules 文件夹即可更新。
yarn根据 yarn.lock 文件判断版本是否是最新的，所以项目是使用npm安装依赖包，更新前要yarn install。
